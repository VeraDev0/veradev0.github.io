My Projects

ScrapComputers

ScrapComputers is a mod for Scrap Mechanic which adds Lua Computers to the game. Unlike other mods, its one of the most performant and unrestrictive mod out there while still being balanced.


Even after facing hostile tactics from certain competitors, including review bombing and hijacking our website's traffic, we still pushed forward to create one of the most powerful computer mods in the Scrap Mechanic community.

sm.regui

Imagine this, you've literally remade MyGui inside MyGui, and somehow the inner one works better than the outer one. That's what sm.regui is


sm.regui is an awesome GUI library for Scrap Mechanic which uses an exploit to write XML code via Lua to create MyGui Layout Files. It has features that MyGui itself doesn't have, such as:

  • Creating, destroying and modifying widgets

  • Custom Font Renderer

  • Fullscreen GUI system


There are more features however you should check it's GitHub out for that.


Steam page Website

VSMX

VSMX is a massive project of mine, which is basically an SDK for Scrap Mechanic, allowing modders to do whatever they please. VSMX will let you:

  • Control the renderer

  • Perform complicated tasks with Multi-threading

  • Full-blown GUI library (ImGui & sm.regui reimplementations for GUIs)


There will be more features that VSMX would have however its private due to compeititors. VSMX will be finished around 2026 or 2027 (this is not a promise)
